# 21is63-software-testing
software testing projects for 3rd year engineering students 
E-commerce Website with User Authentication
This project is an e-commerce website similar to Amazon, featuring user authentication and product management functionalities. The backend is built with Node.js, Express, and SQLite, while the frontend is developed using HTML, CSS, and JavaScript.

ecommerce-clone is the actual code with no errors
cart_error is the code with errors in adding items to the cart
login_error is the code where no matter how much you try you cannot login 

Features
User Registration and Login
Product Listing and Search
Cart Management
Checkout Process
Prerequisites
Node.js installed on your machine
NPM (Node Package Manager)
Git (optional, for cloning the repository)

--------------------STEPS TO RUN--------------------


Installation
1: Clone the repository:
git clone https://github.com/KingAhmedii/21is63-software-testing.git
cd ecommerce-clone
2: Install the dependencies:
npm install
3: Start the backend server:
npm start
4: Open the application in your browser:

Open index.html, login.html, and register.html in your web browser.

Project Structure
server.js: Backend server implementation using Express and SQLite.
index.html: Home page displaying product listings.
login.html: Login page for user authentication.
register.html: Registration page for creating new accounts.
cart.html: Cart page displaying items added to the cart.
styles.css: CSS file for styling the application.
script.js: JavaScript file for frontend functionality.
database.sqlite: SQLite database file (automatically created).
Usage
Register: Create a new account on the registration page.
Login: Log in with your credentials on the login page.
Browse Products: View and search for products on the home page.
Add to Cart: Add products to your cart.
View Cart: View the items in your cart and proceed to checkout.
